# wgpu-viz-prototype
Prototype for very fast visualizations in the browser using WebGPU/Vulkan for extremely fast and efficient GPU rendering.

# Installation

1. Install `pygfx`
2. install `jupyter_rfb` https://github.com/vispy/jupyter_rfb
3. Install `caiman`
4. Install `jupyterlab`
